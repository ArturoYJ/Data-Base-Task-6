# Sistema de Reportes de Biblioteca (Docker + Next.js + PostgreSQL)

Este proyecto es una aplicación web para visualizar reportes y métricas clave de una biblioteca. Utiliza Vistas SQL optimizadas para el análisis de datos y una interfaz moderna en Next.js.

## Requisitos Previos

- **Docker Desktop** instalado y corriendo.
- No necesitas instalar Node.js ni PostgreSQL localmente.

## Cómo Iniciar el Proyecto

1. Clona el repositorio o descarga la carpeta.
2. Abre tu terminal en la carpeta principal del proyecto.
3. Ejecuta el comando mágico:
   ```bash
   docker compose up --build
   ```
